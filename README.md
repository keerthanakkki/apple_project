This project is a responsive website design dedicated to the Apple Vision Pro. It showcases modern web development practices including responsive design, smooth animations, and user-friendly interface. The website is optimized for all device sizes, from mobile phones to desktop screens.

## 🚀 Installation

### Prerequisites
- Git installed on your system
- A modern web browser
- (Optional) Python 3 or Node.js for local server

### Steps

**1. Clone the repository:**
```bash
git clone https://github.com/codebynikhil-cpp/Responsive-Apple-Vision-Website-Design.git
```

**2. Navigate to the project directory:**
```bash
cd Responsive-Apple-Vision-Website-Design
```

**3. Open the project:**

**Option A: Direct in browser**
- Simply open `index.html` in your web browser

**Option B: Using Python local server**
```bash
# Python 3
python -m http.server 8000
```

**Option C: Using Node.js http-server**
```bash
npx http-server
```

**4. Access the website:**
- Direct: Open `index.html` in browser
- Local server: `http://localhost:8000` (or port shown in terminal)
